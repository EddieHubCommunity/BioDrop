## Moving from JSON to Forms

If you have already created your Profile with JSON, and now have started using Forms to make more changes, you need to remove your JSON file.

<Alert
  type="warning"
  message={
    <>
      <strong>NOTE: </strong>
      Make sure you have made at least 1 update to your Profile using Forms <b>
        before you remove your JSON file
      </b>.
    </>
  }
/>

1. Log into your GitHub Account
2. Go to your Repositories

![go to repositories](https://user-images.githubusercontent.com/82668196/208293646-6fbbabe6-f6c6-429f-aaa4-fc77876ce691.png)

3. Then click on the `BioDrop` repo (fork)

![BioDrop repo](https://user-images.githubusercontent.com/82668196/208293648-efd3e4ef-2094-4e42-ad33-3e09286c8809.png)

4. Now we need to make sure your fork is up to date. You can do this by clicking on `Sync fork` and click `Update branch` (if available)

![update branch](https://user-images.githubusercontent.com/82668196/208293657-8e0508c7-f8aa-45db-9f98-2ce9662ac1d8.png)

5. Click `Go to file` and type your GitHub username
6. Click on your `json` file

![find json file](https://user-images.githubusercontent.com/82668196/208293664-85ac522b-6fc4-4e79-863f-389d8a337f43.png)

7. Click the `delete` button icon at the top right

![delete json file](https://github.com/EddieHubCommunity/BioDrop/assets/624760/b308cbba-ec63-4e92-b3ed-83ed0e86d01f)

8. Scroll to the bottom and commit new file by adding the commit message that starts `data: ` followed by your GitHub username (for example `data: SaraJaoude`)

You can skip `optional extended message`

9. Click `create new branch` and you can use the default name provided
10. Then click `Propose changes`

![propose changes](https://user-images.githubusercontent.com/82668196/208293672-3c9480d3-9cf0-452c-8f5f-48559246bc2e.png)

11. Click `compare across forks` and select the destination on the left and the source on the right
12. Click `Create pull request`
13. You will see a title and description for the Pull Request, fill these in and click `Create pull request`

![create pull request](https://user-images.githubusercontent.com/82668196/208293677-79bd2bb8-ee23-4aed-bfef-ea438a9ef3c5.png)

14. Now you need to wait to have your Pull Request (changes) reviewed

![pull request](https://user-images.githubusercontent.com/82668196/208293679-822b4d1e-ecc9-49f5-b363-4a0ecf9922f4.png)

An approving review from one of BioDrop's maintainers will show a green check mark next to their id.

![maintainer review](https://user-images.githubusercontent.com/35927536/217060602-ba3678d1-f655-4805-a47d-e40f1ddcd59f.png)

15. You will receive a GitHub notification when you have a comment, review or your Pull Request has been merged
