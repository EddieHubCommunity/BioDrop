## Gitpod

Gitpod is an open-source application that provides a prebuilt development environments in your browser - powered by VS Code.<br />
It automates the provisioning of a ready-to-code developer environment, so you will have all the dependencies you need (for example MongoDB).<br />
With Gitpod, thereâ€™s no need to download any code to your machine. It can spin up a server-side-dev-environment for you in seconds from any Git Repository.<br />

## How to use Gitpod

There are 2 ways you can open a repository with Gitpod:

- Prefix the url with `gitpod.io/#`
- Use the [Browser Extension](https://www.gitpod.io/docs/configure/user-settings/browser-extension)

Setup environment [variables](/docs/environments/environment-variables)
