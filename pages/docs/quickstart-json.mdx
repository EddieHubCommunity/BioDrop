### Add your Profile

There are 4 ways you can add your Profile with json, but for this Quickstart we will use the GitHub UI.

To do this, you need a GitHub account. If you do not have one yet, you can create one for free with an email address and password.

1. Log in to your GitHub Account
2. Visit the <Link target="_blank" href="https://github.com/EddieHubCommunity/BioDrop">BioDrop repo</Link>

![BioDrop GitHub repo](https://user-images.githubusercontent.com/34659530/266604042-6d255d25-8bca-4aed-b2c1-65eb3e133da7.png)

3. If you wish to be kept up to date with the repo, hit that "Star" at the top right

![Start BioDrop](https://user-images.githubusercontent.com/34659530/266605023-04688017-e464-4df7-a74c-ab9b16668fa4.png)

4. Click on Fork

![create a new fork](https://user-images.githubusercontent.com/34659530/266606626-c4191acf-e86e-4375-a0a6-4b0bbd5afc30.png)

![create a new fork](https://user-images.githubusercontent.com/34659530/269885058-6ddcae2c-d067-4c24-8462-b6db79ce2659.png)

5. Accept the default fork creating a copy of the BioDrop project in your account - this will take you to the forked repo

![BioDrop form](https://user-images.githubusercontent.com/34659530/269885676-d2be4b89-6f19-40db-b673-943b23ab9519.png)

6. Click add file and select "create new file"

![Create new file](https://user-images.githubusercontent.com/34659530/269888417-3562f866-2c76-4da8-98bf-cc6112b8d3de.png)

7. In the file path at the top, add `data/` followed by your GitHub username, exactly as it appears including capitalisation, and then append `.json` to it.

For example for username **"ChiragThakur"**:

> - _**Correct**_: `BioDrop/data/ChiragThakur.json` (including capitalisation in filename)
> - _**Incorrect**_: `BioDrop/data/chiragthakur.json` (filename capitalisation does not match username capitalisation)

![create json file](https://user-images.githubusercontent.com/34659530/269888445-8b6895b3-f570-4c50-b736-b8235163aa4d.png)

8. For the file contents use the json structure below:

<ClipboardCopy>
```js
{
  "name": "Chirag Thakur",
  "bio": "Lawyer turned **Digital Nomad**. Travel enthusiast Yoga & Peloton aficionado"
}
```
</ClipboardCopy>

| field | type   | description                                                 |
| :---- | :----- | :---------------------------------------------------------- |
| name  | string | Your display name on your Profile                           |
| bio   | string | Your description about yourself (you can use markdown here) |

9. Now add links to your social media and other content (for example: website) so they appear on your Profile.

<ClipboardCopy>
```js
{
  "name": "Chirag Thakur",
  "bio": "Lawyer turned **Digital Nomad**. Travel enthusiast Yoga & Peloton aficionado",
  "links": [
    {
      "name": "Follow me on Twitter",
      "url": "https://twitter.com/CHIRAGTHAKUR95",
      "icon": "FaTwitter"
    }
  ]
}
```
</ClipboardCopy>

_note: do not have duplicate links (url)_

| field | type   | description                                                                                                                                                           |
| :---- | :----- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| name  | string | The display name for the button                                                                                                                                       |
| url   | string | Full url to the resource                                                                                                                                              |
| icon  | string | The icon you would like displayed. We support the following icon sets: **Fa** and **Si** from React Icons. You can search available [icons](https://biodrop.io/icons) |

![propose new file](https://user-images.githubusercontent.com/34659530/269891349-815771e1-6548-485d-9032-697f98265d63.png)

To add more links, add another object inside the links collection. For example:

<ClipboardCopy>
```js
  "links": [
    {
      "name": "Follow me on Twitter",
      "url": "https://twitter.com/CHIRAGTHAKUR95",
      "icon": "FaTwitter"
    },
    {
      "name": "Follow me on Instagram",
      "url": "https://www.instagram.com/chiragthakur9/",
      "icon": "FaInstagram"
    }
  ]
```
</ClipboardCopy>

10. Scroll to the bottom and commit new file by adding the commit message that starts `data: ` followed by your GitHub username (for example `data: ChiragThakur`)

You can skip `optional extended message`

<Alert
  type="info"
  message={
    <>
      <strong>NOTE</strong>: Please use our online{" "}
      <Link href="https://biodrop.io/playground">Playground</Link> tool to check
      your JSON file is <i>valid</i> before committing your pull request.
    </>
  }
/>

11. Click `create new branch` and you can use the default name provided
12. Then, click `Propose new file`

![propose new file](https://user-images.githubusercontent.com/34659530/269893941-338a9991-8023-4b99-b7a7-aa9c29bae9a8.png)

<Alert
  type="warning"
  message={
    <>
      13. <strong>HIGHLY IMPORTANT</strong>: Click compare across forks and you
      will see 4 drop downs
    </>
  }
/>

| Dropdown     | Description                          | Example values              |
| :----------- | :----------------------------------- | :-------------------------- |
| 1st dropdown | Base repository (destination)        | `EddieHubCommunity/BioDrop` |
| 2nd dropdown | Base repository branch (destination) | `main`                      |
| 3rd dropdown | Your repository (source)             | `your-username/BioDrop`     |
| 4th dropdown | Your repository branch (source)      | `your-username-patch-1`     |

The first dropdown for the `base` repository (destination) should be set to `EddieHubCommunity/BioDrop` not your username `your-username/BioDrop`.

![compare across forks](https://user-images.githubusercontent.com/34659530/269904006-12c06eb3-8fcb-420d-ac09-93a4dd50d1be.png)

14. Click `Create pull request`
15. The description will be filled with a template, please fill in the appropriate sections

![create pull request](https://user-images.githubusercontent.com/82668196/208292192-704402e7-c2d6-432e-8dab-cb6a678866c1.png)

16. Click `Create pull request`

17. Now you need to wait to have your Pull Request (changes) reviewed

![pull request](https://user-images.githubusercontent.com/34659530/269924986-4ca0783c-1fe6-4331-bbde-b48876bd9bf2.png)

An approving review from one of BioDrop's maintainers will show a green check mark next to their id.

![maintainer review](https://user-images.githubusercontent.com/34659530/269925671-25785be9-7b96-46a7-afc3-f5a38a8fbf90.png)

18. You will receive a GitHub notification when you have a comment, review, or your Pull Request has been merged
19. Once merged, your profile will be available a few minutes later on your custom URL (<Link href="https://biodrop.io/SaraJaoude">BioDrop.io/SaraJaoude</Link>)

![merged pull request](https://user-images.githubusercontent.com/34659530/269926139-80038d45-ab70-42ea-b43f-14bd37e0a5ff.png)
